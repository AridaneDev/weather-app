# 🌤️ Weather App

Aplicación del tiempo desarrollada en React + TypeScript + Tailwind CSS.
Muestra el clima actual de cualquier ciudad con fondo dinámico según las condiciones meteorológicas.

## 🚀 Características

- Buscador de ciudades en tiempo real
- Temperatura actual en °C
- Descripción del clima en español
- Humedad y velocidad del viento
- Fondo dinámico según el clima (soleado, nublado, lluvia, nieve...)

## 🛠️ Tecnologías usadas

- ⚛️ React 19
- 🛠 TypeScript
- ⚡ Vite
- 🎨 Tailwind CSS 4
- 🌦️ OpenWeatherMap API
